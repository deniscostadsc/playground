Read an integer N that is the number of test cases.
Each test case is a line containing two integer numbers
X and Y. Print the sum of all odd values between them,
not including X and Y.

Input

The first line of input is an integer N that is the number
of test cases that follow. Each test case is a line
containing two integer X and Y.

Output

Print the sum of all odd numbers between X and Y.
